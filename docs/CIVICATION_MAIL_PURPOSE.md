# Civication Mail Purpose

Civication Mail for næringsliv skal simulere arbeidshverdagen, ikke generiske karrieremeldinger.

Hver mail må beskrive en konkret jobbsituasjon med:
1. rolle og kontekst
2. oppgave
3. friksjon/presselement
4. handlingsvalg
5. kompetanse i bruk
6. konsekvens i drift, tillit eller tempo

Mål: spilleren trener praktisk dømmekraft i butikkarbeid gjennom realistiske, korte scenarioer.
