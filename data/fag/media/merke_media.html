MEDIA – full teoretisk beskrivelse

Dette er den interne teoriteksten for merket MEDIA i History Go. Feltet handler om presse, journalistikk, redaksjoner, offentlighet, nyheter, plattformer, mediehus, ytringsfrihet, kildekritikk, informasjon, propaganda, algoritmer, publisering og steder der samfunnet ser, tolker og diskuterer seg selv.

1. Felt

Media er samfunnets organiserte informasjons- og offentlighetsfelt. Det handler om hvordan nyheter, bilder, kommentarer, dokumentasjon, debatt, data, lyd, tekst og digitale strømmer produseres, redigeres, distribueres og tolkes.

I History Go viser media-merket hvordan steder kan være knyttet til aviser, redaksjoner, trykkerier, kringkasting, pressehistorie, offentlig debatt, medieteknologi, plattformer, journalistikk, sensur, propaganda, kildearbeid og den moderne informasjonsbyen.

2. Normativ dimensjon

Feltet bygger ofte på idealer som:

  • ytringsfrihet
  • pressefrihet
  • kildekritikk
  • åpenhet
  • etterprøvbarhet
  • redaksjonell uavhengighet
  • opplyst offentlighet
  • ansvarlig publisering

Media skal informere offentligheten, men mediefeltet er også preget av makt: hvem som setter dagsorden, hvilke saker som blir synlige, hvem som får stemme, og hvordan økonomi, teknologi og algoritmer former hva publikum faktisk møter.

3. Doxa

«Media bare formidler det som skjer.»

Doxa skjuler at media også velger, vinkler, sorterer, prioriterer, rammer inn, publiserer, arkiverer, modererer og kommersialiserer virkeligheten. Media er ikke bare en kanal, men et sett av institusjoner, teknologier, rutiner og maktformer.

4. Metodisk dimensjon

  • pressehistorisk analyse
  • redaksjons- og institusjonsanalyse
  • kildekritisk analyse
  • diskurs- og framinganalyse
  • plattform- og algoritmeanalyse
  • offentlighetsanalyse
  • propaganda- og desinformasjonsanalyse
  • medieøkonomisk analyse

5. Materiell dimensjon

Media hviler på konkrete ting og steder:

  • redaksjoner, avishus, trykkerier, TV-/radiohus og presseklubber
  • kameraer, mikrofoner, notatblokker, desker, servere, datasystemer og publiseringsverktøy
  • forsider, overskrifter, arkiv, pressefoto, kommentarspalter og nyhetsbrev
  • plattformer, algoritmer, feed-strukturer, søk, lenker, moderatorverktøy og annonsesystemer
  • pressekonferanser, rettssaler, politiske møter, demonstrasjoner og andre nyhetssteder

6. Sosial dimensjon

  • journalister, redaktører, fotografer, kommentatorer, deskere og utviklere
  • kilder, varslere, eksperter, politikere, organisasjoner og publikum
  • redaksjonelle rutiner, kildevern, pressekritikk og medieetikk
  • debattfelt, kommentarfelt, sosiale medier, deling og nettverkseffekter
  • konflikter om tillit, eierskap, makt, sensur, feilinformasjon og kommersiell styring

Media er et sosialt system for synlighet. Det bestemmer ikke bare hva som blir sagt, men også hvor fort det spres, hvem som blir trodd, og hvilke hendelser som blir stående som offentlige fakta.

7. Geografisk / romlig dimensjon

Media er knyttet til steder: redaksjonsbygg, aviskvartaler, presseklubber, trykkerier, kringkastingshus, offentlige institusjoner, rettssaler, rådhus, demonstrasjonssteder og digitale infrastrukturer.

I History Go gjør dette byen lesbar som mediekart: hvor nyheter produseres, hvor makt møter presse, hvor bilder tas, hvor debatter oppstår, og hvor offentligheten får sine faste rom.

8. Temporal / historisk dimensjon

  • fra trykkpresse og aviskvartaler til radio, TV, nettaviser og plattformer
  • fra redaksjonelle rytmer til kontinuerlige nyhetsstrømmer
  • fra papirforsider til pushvarsler, feeds og algoritmisk distribusjon
  • fra nasjonale offentligheter til fragmenterte digitale deloffentligheter
  • fra arkiv og dokumentasjon til søk, data, metadataspor og digital glemsel

Mediehistorien er en historie om tempo. Nyheter går fra å være periodiske hendelser til å bli et kontinuerlig miljø folk lever inne i.

9. Blindsoner

  • redaksjonelle arbeidere, fotografer, deskere og utviklere bak synlige bylines
  • lokalaviser, nisjemedier og minoritetsmedier som forsvinner bak riksmedier
  • kildenes makt og kildeavhengighet i nyhetsproduksjon
  • algoritmisk distribusjon og plattformmakt som skjules bak brukeropplevelsen
  • økonomiske føringer, annonsemodeller og eierskap
  • feil, rettelser, pressekritikk og usynlige redaksjonelle valg

11. Bidrag fra steder og personer

  • viser hvordan offentligheten produseres gjennom redaksjoner, kilder og publisering
  • kobler avishus, TV-/radiohus, presseklubber og nyhetssteder til konkret mediehistorie
  • gjør journalistisk arbeid, kildearbeid, presseetikk og mediemakt synlig
  • viser hvordan digitale plattformer endrer hva som blir sett og delt
  • skiller media som offentlighets- og informasjonsfelt fra Film & TV, populærkultur og politikk

MEDIA-merket gjør det mulig å lese byen gjennom offentlighetens maskineri — gjennom stedene der saker oppstår, kilder møter journalister, redaksjoner velger, og samfunnet får vite hva det mener at det vet.
