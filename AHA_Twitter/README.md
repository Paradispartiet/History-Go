# AHA Twitter (Project Foundation)

Dette er grunnoppsettet for en Next.js-app med TypeScript, Tailwind og Supabase-scaffolding.

## Innhold i foundation-commit
- Next.js + TypeScript + Tailwind baseline
- Mappestruktur for `app/`, `components/`, `lib/`, `types/` og `supabase/migrations/`
- `AppShell` og en tom `FeedPage`
- Konstanter for post-typer og node-typer
- Grunnleggende TypeScript-typer
- Supabase klient/server scaffold
- SQL-migrasjoner for schema, RLS og triggers

## Kom i gang
```bash
npm install
npm run dev
```

## Miljøvariabler
Kopier `.env.example` til `.env.local` og fyll inn verdier.
